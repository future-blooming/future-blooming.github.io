Small Tips to help you better




    Enter any question or paragraph on the left hand side. The right hand pane gives you the instant output classifying your input into the different cognitive levels of Bloom's Taxonomy.

    Refer to the image aside for the colour coding of the different domains.

    For more details about Bloom's Taxonomy read the Wikipedia Page on Blooms Taxonomy. For more details about the project you can visit this page
    The original Bloom's Classification of the cognitive domain consisted of the following six levels:
  • Knowledge
  • Comprehension
  • Application
  • Analysis
  • Synthesis
  • Evaluation
    These are arranged in that order to indicate the gradual change in the type of cognitive sense required to understand the problem.
